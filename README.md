Project Name:
 To-Do List 

Description:
The Flask To-Do List Web Application is a simple web-based tool that allows users to register, log in, and manage a to-do list. Users can create, edit, and delete tasks, with each task having a title, category, priority, status, and due date. The app ensures that each user's tasks are stored in a secure and personal database, providing a convenient way to stay organized and track progress.

Purpose:
The purpose of this project is to help users efficiently manage their tasks by offering an intuitive interface for adding, updating, and organizing tasks based on priority, category, and due date. It also serves as a practical demonstration of user authentication, session management, and interaction with a database in a web application.

Value:
This project adds value by offering a functional tool that can help users stay organized and prioritize their tasks, which is useful for both personal and professional task management. It provides a hands-on example of how to build a full-stack web application using Flask, covering essential concepts such as user authentication, task management, and database integration. It also demonstrates best practices in securing user data, such as password hashing, and efficiently managing database relationships using SQLAlchemy.

Technologies Used:
The project utilizes several key technologies to provide a seamless user experience and robust functionality. Flask is the backbone of the application, a lightweight Python web framework that handles routing, sessions, and requests. It enables efficient web server creation and management. Flask-SQLAlchemy is an extension integrated into Flask to simplify database management through SQLAlchemy, making it easy to define models, perform queries, and handle database interactions. The application uses SQLite, a lightweight, file-based relational database system, to securely store user data and tasks. Werkzeug, a utility library, is employed for securely hashing passwords, ensuring that user authentication is both effective and safe. On the front end, HTML, CSS, and JavaScript are used to create dynamic, interactive, and responsive web pages. These technologies work together to build a full-stack web application, handling everything from user input to data storage and display.
